# 1. when using a list becomes painful

## 🎯 Learning Objectives
*   when to use a dictionary v.s. list

## 📝 Notes
*   - **As a list**:
    
    - Just a bunch of notes in the order you created them.
    - Question it answers well: “Show me my notes in the order I wrote them.”
- **As a dictionary**:
    
    - Keys could be titles, IDs, or dates → values are the note contents.
    - Question it answers well: “Given this **title/ID/date**, show me that specific note.”

So even with notes, it comes back to the same rule:

> If you mostly care about **order** → list.  
> If you mostly care about looking things up by a **label** → dictionary.
