## 📝 Notes
*   **1. What a dictionary is**

> A **dictionary** is a collection of key–value pairs, like labeled drawers.

Example:

Python

```python
person = {
    "name": "Tylor",
    "age": 40,
    "track": "Software Engineering"
}
```

- Keys: `"name"`, `"age"`, `"track"`
- Values: `"Tylor"`, `40`, `"Software Engineering"`

**2. Creating a dictionary**

Literal syntax:

Python

```python
hero = {
    "name": "Rambo",
    "power": "gun-fighting"
}
```

Empty, then fill later:

Python

```python
data = {}
data["key"] = "value"
```

**3. Reading values**

Unsafe (can crash if key missing):

Python

```python
person["name"]          # OK if "name" exists
person["middle_name"]   # KeyError if missing
```

Safe:

Python

```python
person.get("name")                  # value or None
person.get("nickname", "No nickname set")  # value or default
```

**4. Adding and updating**

Same syntax for both — depends on whether the key already exists:

Python

```python
hero["universe"] = "Marvel"   # add new key
hero["power"] = "gun-slinger" # update existing key
```

If key doesn’t exist → added  
If key exists → overwritten

**5. Error types to remember**

- Lists → bad index → **IndexError**
- Dictionaries → missing key → **KeyError**
