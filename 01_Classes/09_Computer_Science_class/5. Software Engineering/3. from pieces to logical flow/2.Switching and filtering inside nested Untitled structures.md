# Switching and filtering inside nested Untitled structures

## 📌 Overview
*   **Course:** SE
*   **Lecture Date:** 2026-04-30 
*   **Instructor:** Maestro

## 🎯 Learning Objectives
*   iterating through double nested structures to filter results

## 📝 Notes
*   **1. The data shape you worked with**

You worked with a **list of dictionaries**:

```python
characters = [
    {"name": "Aegis", "class": "warrior", "level": 12},
    {"name": "Lyra", "class": "mage", "level": 8},
    {"name": "Korr", "class": "warrior", "level": 5},
]
```

- The **list** holds multiple records.
- Each **dictionary** is one record with fields like `"name"`, `"class"`, `"level"`.

---

**2. The filter loop pattern**

You used this pattern over and over:

```python
matches = []  # 1) start with empty results

for record in records:  # 2) loop through every record
    if some_condition_about(record):  # 3) decide match vs skip
        matches.append(record)        # 4) collect the match
```

Roles:

- `if ...` → **matching logic** (decides yes/no)
- `.append(...)` → **collection logic** (actually adds to the result list)

---

**3. Different kinds of conditions you used**

You filtered on:

- **Equality**
    
    ```python
    if character["class"] == "mage":
    ```
    
- **Inequality / negation**
    
    ```python
    if character["class"] != "warrior":
    ```
    
- **Numeric range (inclusive)**
    
    ```python
    if character["class"] == "warrior" and character["level"] >= 5 and character["level"] <= 12:
    ```
    
- **Threshold**
    
    ```python
    if character["level"] > 10:
    ```
    

Each time, same structure: change only the condition, keep the loop + append pattern.

---

**4. Search loop vs filter loop**

- **Filter loop** → goes through _all_ items and collects **every** match in a list.  
    That’s what you used the whole lesson.
    
- **Search loop** → usually stops when it finds the _first_ match (with `break` or `return`).
    

You can now clearly recognize that:

```python
matches = []
for character in characters:
    if character["class"] == "warrior":
        matches.append(character)
```

is a **filter loop**.
