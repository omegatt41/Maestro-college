# 7.Breaking repetition into reusable steps

## 📌 Overview

## 🎯 Learning Objectives
*   Breaking Repetition into reusable steps

## 📝 Notes
*   **Core idea**

> **Keep the loop outside, put the “work for one item” in a helper function.**  
> Loop = _“for each thing…”_  
> Helper = _“what to do with one thing.”_

**Pattern: loop helper**

Example shape:

Python

```python
def process_item(item):
    # all the logic for ONE item
    # clean / validate / transform
    return result_or_none

items = [...]
results = []

for item in items:
    result = process_item(item)
    if result is not None:      # loop decides whether to keep it
        results.append(result)

print(results)
```

- Helper runs **once per item**.
- Loop stays short and readable.

**What belongs in the helper**

- Stripping, formatting, transforming one value  
    `name.strip()`, `name.capitalize()`, `score * 10`
- Validating that one value  
    `if len(cleaned) < 3: return None`
- Anything that does **not** need the whole list or the index.

Think: **“Do one focused thing to one item.”**

**What belongs in the loop**

- Iteration and control:
    - `for name in players:`
    - `for i, name in enumerate(players):`
- Collecting results:
    - `results.append(result)`
- Using index / position:
    - `print(i, "=>", result)`
- Deciding what to _do_ with helper’s return:
    - `if result is not None: ...`

The loop should read like:

> “Go through everything, call helper, keep or skip.”

**Why this is good**

- **Clarity:** loop is short; you can read it like English.
- **Reuse:** same helper can be used in other loops or places.
- **Less bugs:** validation lives in one place, not copy‑pasted.

**Heuristic**

- If your loop body is more than **5–7 lines** or repeats similar logic: → **Extract a helper** that handles one item.

before:
```python
def clean_name(raw_name):
    cleaned = raw_name.strip()
    cleaned = cleaned.capitalize()
    return cleaned

players = ["  tYlOr", "  aLEX  ", "jo  "]

cleaned_players = []
for i, name in enumerate(players):
    result = clean_name(name)
    print(i, "=>", result)
    cleaned_players.append(result)

print(cleaned_players)
```
