## 📝 Notes
*  **Overall feature**

- Input: list of **order dicts**  
    `{"id", "customer", "status", "amount"}`
- Output: **sorted report** of customers and their total paid amount, highest first.

---

**Pipeline (functions calling functions)**

1. `filter_paid_orders(orders)`
    
    - Loop over `orders`
    - Keep only orders where `status == "PAID"`
    - Return `paid_orders` (list of dicts)
2. `summerize_paid_orders(orders)`
    
    - Input: only paid orders
    - Build a new list of **simplified** dicts:  
        `{"customer": ..., "amount": ...}`
    - Return `summary_list`
3. `group_customer_totals(summerized_orders)`
    
    - Use `customer_totals = {}` as an **accumulator**
    - For each simplified order:
        - If customer already in dict → add amount
        - Else → start their total
    - Convert `customer_totals` into list of dicts:  
        `{"customer": name, "total_amount": total}`
    - Return `grouped_list`
4. `manual_sort_customers(total_list)`
    
    - Bubble sort on `["total_amount"]`
    - Sort **descending** (bigger totals first)
    - Return sorted `total_list`
5. `main()`
    
    - Creates `orders`
    - Runs the full pipeline:
        - `paid_list = filter_paid_orders(orders)`
        - `summary_list = summerize_paid_orders(paid_list)`
        - `totals = group_customer_totals(summary_list)`
        - `sorted_totals = manual_sort_customers(totals)`
    - Prints final report: customer name + total.

---

**Acceptance-style behavior**

- Only `"PAID"` orders are counted.
- All paid orders for the same customer are **summed**.
- Customers with no paid orders never appear.
- Final list is sorted by `total_amount` in **descending** order. 

## 💻 Code Examples / Algorithms
```python
def filter_paid_orders(orders):
    paid_orders = []
    for order in orders:
        if order["status"] == "PAID":
            paid_orders.append(order)
    return paid_orders

def summerize_paid_orders(orders):
    summary = []
    for order in orders:
        simplified = {
            "customer": order["customer"],
            "amount": order["amount"]
        }
        summary.append(simplified)
    return summary
  
def group_customer_totals(summerized_orders):
    customer_totals = {}
    for order in summerized_orders:
        name = order["customer"]
        amount = order["amount"]
        if name in customer_totals:
            customer_totals[name] += amount
        else:
            customer_totals[name] = amount
    grouped_list = []
    for name, total in customer_totals.items():
        grouped_list.append({"customer": name, "total_amount": total})
    return grouped_list

def manual_sort_customers(total_list):

    # Manual Bubble Sort implementation
    n = len(total_list)
    for i in range(n):
        for j in range(0, n - i - 1):
            # Compare current item total with the next item total
            if total_list[j]["total_amount"] < total_list[j + 1]["total_amount"]:
                # Swap if the next item is larger (for descending order)
                total_list[j], total_list[j + 1] = total_list[j + 1], total_list[j]
    return total_list

def main():
    orders = [
        {"id": 1, "customer": "Alice", "status": "PAID", "amount": 50},
        {"id": 2, "customer": "Bob", "status": "CANCELLED", "amount": 30},
        {"id": 3, "customer": "Alice", "status": "PAID", "amount": 20},
        {"id": 4, "customer": "Charlie", "status": "PAID", "amount": 100}
    ]

    paid_list = filter_paid_orders(orders)
    summary_list = summerize_paid_orders(paid_list)
    totals = group_customer_totals(summary_list)

    # Using the manual sort instead of sorted() with lambda
    sorted_totals = manual_sort_customers(totals)
    print("Final report (descending order):")
    for entry in sorted_totals:
        print(f"Customer: {entry['customer']}, Total: {entry['total_amount']}")
if __name__ == "__main__":
    main()
```
