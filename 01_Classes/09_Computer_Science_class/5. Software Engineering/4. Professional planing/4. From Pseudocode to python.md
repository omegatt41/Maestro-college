## 🎯 Learning Objectives
*   From pseudocode to python

## 📝 Notes
*   **Core idea**

> One line of pseudocode → one line (or block) of Python, in the same order, with matching indentation.

Python should look like a **mirror** of the pseudocode structure.

**Common mappings**

- `ask user for X` → `x = input("Enter X: ")`
- `show "Message"` → `print("Message")`
- `IF condition` → `if <condition>:`
- `ELSE IF condition` → `elif <condition>:`
- `ELSE` → `else:`
- `END IF` → end of the indented block (Python uses indentation, not `END`)

**Comparison operators (critical)**

Pseudocode → Python:

- `is` → `==`
    - `IF x is 10` → `if x == 10:`
- `greater than or equal to` → `>=`
    - `IF score is greater than or equal to 90` → `if score >= 90:`

Big rule:

- `=` assignment (store a value)
- `==` comparison (check a value)

Never use `=` in an `if` condition.

**Input handling**

- `input()` returns a **string**
- For numeric comparisons, convert:

Python

```python
score = int(input("Enter score: "))
```

Then conditions like `score >= 90` are valid.

**Indentation & structure**

From pseudocode:

- Lines under `IF` / `ELSE IF` / `ELSE` are **indented** in Python.
- You must keep **all branches** from pseudocode:
    - `IF` …
    - `ELSE IF` …
    - `ELSE` …

Example:

Pseudocode:

- IF password is "secret"
-   show "Access granted"
- ELSE
-   show "Access denied"

Python:

Python

```python
if password == "secret":
    print("Access granted")
else:
    print("Access denied")
```

**Grading example from the lesson**

Pseudocode:

- ask user for score
- IF score is ≥ 90 → show "A"
- ELSE IF score is ≥ 80 → show "B"
- ELSE → show "Keep trying"

Python:

Python

```python
score = int(input("Enter score: "))
if score >= 90:
    print("A")
elif score >= 80:
    print("B")
else:
    print("Keep trying")
```
