# 4. Counters,score tables, and grouping

## 📌 Overview
*   **Course:** grouping and counting in dictionaries
*   **Lecture Date:** 2026-04-18 
*   **Instructor:** maestro

## 🎯 Learning Objectives
*   Counting → default `0`  
*   Grouping → default `[]`

- use a dict to **count per key** with get(key, 0) then += 1
		
	names = ["alice", "bob", "alice"]
	
	counts = {}
	for name in names:
	    current = counts.get(name, 0)  # 0 if not there yet
	    counts[name] = current + 1

- use a dict to **group per key** with get(key, []) then            .append(key)
	
	words = ["hello", "hi", "hey", "world", "who", "hat", "wow"]
	
	groups = {}
	for word in words:
	    first_letter = word[0]
	    current_list = groups.get(first_letter, [])
	    current_list.append(word)
	    groups[first_letter] = current_list
