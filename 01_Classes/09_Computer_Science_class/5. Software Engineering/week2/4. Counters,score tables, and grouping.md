# 4. Counters,score tables, and grouping

*   **Course:** grouping and counting in dictionaries
*   **Lecture Date:** 2026-04-18

## 🎯 Learning Objectives
*   Counting → default `0`  
*   Grouping → default `[]`

- use a dict to **count per key** with get(key, 0) then += 1
		
	names = ["alice", "bob", "alice"]
	
	counts = {}
	for name in names:
	    current = counts.get(name, 0)  # 0 if not there yet
	    counts[name] = current + 1

- use a dict to **group per key** with get(key, []) then            .append(key)
	
	words = ["hello", "hi", "hey", "world", "who", "hat", "wow"]
	
	groups = {}
	for word in words:
	    first_letter = word[0]
	    current_list = groups.get(first_letter, [])
	    current_list.append(word)
	    groups[first_letter] = current_list

## 📝 Notes
* **1. Dictionary as a scoreboard (counting pattern)** 🧮

> Pattern: “counter per key”

Used when you want: **“How many times does each thing appear?”**

Core idea in a loop:

Python

```python
current = counts.get(key, 0)   # default 0
counts[key] = current + 1
```

Example:

Python

```python
names = ["alice", "bob", "alice"]

counts = {}
for name in names:
    current = counts.get(name, 0)
    counts[name] = current + 1

# {'alice': 2, 'bob': 1}
```

- Use this when values should be **integers that increase**.
- Safe from `KeyError` because of `get(key, 0)`.

**2. Dictionary for grouping (accumulation pattern)** 🧺

> Pattern: “growing list per key”

Used when you want: **“Which items belong under each group?”**

Core idea in a loop:

Python

```python
current_list = groups.get(key, [])  # default empty list
current_list.append(value)
groups[key] = current_list
```

Example (group by first letter):

Python

```python
words = ["dog", "dove", "cat", "cow", "ant"]

groups = {}
for word in words:
    first_letter = word[0]
    current_list = groups.get(first_letter, [])
    current_list.append(word)
    groups[first_letter] = current_list

# {'d': ['dog', 'dove'], 'c': ['cat', 'cow'], 'a': ['ant']}
```

- Use this when values should be **lists that grow**.
- Safe from `KeyError` because of `get(key, [])`.

**3. When to use which default** ⚙️

- **Counting:**
    
    - values = integers
    - `get(key, 0)` then `+ 1`
- **Grouping:**
    
    - values = lists
    - `get(key, [])` then `.append(...)`

**4. Typical patterns you used** ✅

- Count words:

Python

```python
word_counts = {}
for word in words:
    current = word_counts.get(word, 0)
    word_counts[word] = current + 1
```

- Group by first letter:

Python

```python
groups = {}
for word in words:
    first_letter = word[0]
    current_list = groups.get(first_letter, [])
    current_list.append(word)
    groups[first_letter] = current_list
```
