# Iterating over dict-keys,values,and items in action

*  looping over dictionaries 

## 📝 Notes
* **. Basic idea: looping over a dict**

Given:

Python

```python
airports = {
    "LAX": "Los Angeles",
    "JFK": "New York",
    "DFW": "Dallas"
}
```

- `for code in airports:` → loops over **keys** (`"LAX"`, `"JFK"`, `"DFW"`).
- This is the same as `for code in airports.keys():`.

**2. Looping over keys**

Use when you only need the **labels** (keys):

Python

```python
for key in my_dict:
    print(key)

# or explicitly
for key in my_dict.keys():
    print(key)
```

You can still get the value if needed:

Python

```python
for key in my_dict:
    value = my_dict[key]
```

**3. Looping over values**

Use when you only care about the **values**, not the keys:

Python

```python
for value in my_dict.values():
    print(value)
```

Example:

Python

```python
prices = {"apple": 1.5, "banana": 0.75}
for price in prices.values():
    print(price)
```

**4. Looping over key–value pairs (`.items()`)**

Use when you need **both** key and value together:

Python

```python
for key, value in my_dict.items():
    print(key, "->", value)
```

Example:

Python

```python
books = {
    "Dune": 412,
    "It": 1138,
    "Foundation": 255
}

for title, pages in books.items():
    print(title, "has", pages, "pages")
```

**5. Quick decision rule (burn this in)**

- Need **only keys** → `for k in d:` (or `d.keys()`)
- Need **only values** → `for v in d.values():`
- Need **both together** → `for k, v in d.items():`
