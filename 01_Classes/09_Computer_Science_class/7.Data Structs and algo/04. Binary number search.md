# 04. Binary number search

## 📌 Overview
- **Definition**: Binary Search → Biary search is a search method for sorted lists where we repeatably check the middle item for the target and cut the search area in half each time
- **Binary Search only works in correctly sorted data**-
	- comparing target to sorted middle can identify safley which half of data we dont need
	- if data isnt sorted target could be anywhere and cannot safley remove any data
- **Core Goal**: to search a sorted number list for  specific number faster
- **Prerequisites**: [[Binary Search]][[O(log n)]]

## ⚙️ Logic & Implementation
1. sort list
2. check middle for target
3. if > target cut right
4. if < target cut left
5. till target found or determined non-exist
### Key Variables:
	- Low = 0
	- Mid = (low + high) // 2
	- High = len(nums) - 1

### Python Implementation
```python
def binary_search(nums, target):
    low = 0
    high = len(nums) - 1
    while low <= high:
        mid = (low + high) // 2
        if nums[mid] == target:
            return True
        elif target < nums[mid]:
            high = mid -1
        else:
            low = mid + 1
    return False
print(binary_search([3, 6, 9, 14, 18, 21], 14))
print(binary_search([3, 6, 9, 14, 18, 21], 7))
```
