# Bubble Sort Notes

**Bubble sort** is a sorting method that repeatedly compares **neighboring elements** in a list.

If the left value is bigger than the right value, bubble sort **swaps** them.

> Bubble sort repeatedly swaps out-of-order neighboring elements until the list becomes sorted.

Core Idea

Bubble sort works by moving larger values toward the right side of the list.

That is why it is called “bubble” sort — larger values gradually _bubble up_ toward the end.

Example:

Python

```python
[5, 1, 4]
```

First comparison:

Python

```python
5 and 1
```

Since `5 > 1`, swap them:

Python

```python
[1, 5, 4]
```

Next comparison:

Python

```python
5 and 4
```

Since `5 > 4`, swap them:

Python

```python
[1, 4, 5]
```

After one full pass, the largest value has moved to the end.

One Bubble Pass

A **pass** means going through the list from left to right, comparing neighboring pairs.

For this list:

Python

```python
[5, 1, 4]
```

One pass compares:

Python

```python
5 and 1
5 and 4
```

After those swaps, the list becomes:

Python

```python
[1, 4, 5]
```

Important Rule

Bubble sort only swaps when the neighboring values are **out of order**.

For ascending order:

Python

```python
if left_value > right_value:
    swap them
```

If the values are already in the correct order, bubble sort leaves them alone.

Example:

Python

```python
[1, 5, 4]
```

Compare `1` and `5`.

Since `1 < 5`, no swap happens.

Python Swap Example

Python lets us swap two list values like this:

Python

```python
nums[0], nums[1] = nums[1], nums[0]
```

Example:

Python

```python
nums = [5, 1, 4]

if nums[0] > nums[1]:
    nums[0], nums[1] = nums[1], nums[0]

print(nums)
```

Output:

Python

```python
[1, 5, 4]
```

Only the first neighboring pair was checked and swapped.

Simple One-Pass Function

Python

```python
def one_bubble_pass(nums):
    for i in range(len(nums) - 1):
        if nums[i] > nums[i + 1]:
            nums[i], nums[i + 1] = nums[i + 1], nums[i]
    return nums

print(one_bubble_pass([5, 1, 4]))
```

Output:

Python

```python
[1, 4, 5]
```

Why Learn Bubble Sort?

In real Python code, you would usually use built-in sorting tools.

But bubble sort is useful for learning because it shows the mechanics of sorting clearly:

Text

```text
compare neighbors → swap if needed → repeat
```

It helps you understand how a list can be sorted step by step.

Key Takeaway

Bubble sort is:

Text

```text
Compare neighboring elements.
Swap them if they are out of order.
Repeat until the list is sorted.
```
