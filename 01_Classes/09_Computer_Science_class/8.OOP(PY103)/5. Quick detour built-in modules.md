**1. What a module is**

> A **module** is a Python file that groups related code (functions, variables, etc.) so you can reuse it.

- Think of it as a **toolbox**:
    - The file = the toolbox
    - Each function/variable = a tool inside
- Example toolboxes:
    - `random` → random numbers, random choices
    - `time` → time-related functions
    - `math` → math functions (sqrt, trig, etc.)
    - `datetime` → dates and times

Why this matters:

- You don’t rewrite common logic (random numbers, math, dates).
- You just import the toolbox and use what’s already written.

**2. Basic import syntax**

To use a toolbox, you **import** it:

Python

```python
import random
```

- Put imports near the **top** of your file.
- After `import random`, you can access things **inside** `random`.

General pattern:

Python

```python
import module_name
```

Examples:

Python

```python
import random
import time
import math
import datetime
```

**3. Dot notation (`module.name`)**

Once imported, you use **dot notation**:

> `module_name.something` = “go inside this module and use `something`”

Examples:

Python

```python
import random

roll = random.randint(1, 6)   # from random module
```

Python

```python
import time

now = time.time()  # from time module
```

Python

```python
import math

root = math.sqrt(16)  # from math module
```

Python

```python
import datetime

current = datetime.datetime.now()  # from datetime module
```

Key idea:

- Left side of the dot: the **module** (toolbox)
- Right side of the dot: a **function or attribute** inside

**4. Built-in module: `random`**

You used:

Python

```python
import random

roll = random.randint(1, 6)
print("You rolled:", roll)
```

- `random.randint(a, b)`
    
    - Returns a random **integer** `N` such that `a <= N <= b`.
    - Example: `random.randint(1, 20)` → your 20-sided dice.
- `random.choice(sequence)`
    
    - Picks **one random element** from a list, tuple, or string.
    - Example:
        
        ```python
        import random
        
        move = random.choice(["rock", "paper", "scissors"])
        print(move)
        ```
        

Other useful ones (not all used in the lesson, but good to know):

- `random.random()` → float in `[0.0, 1.0)`
- `random.shuffle(list)` → shuffles a list **in place**

**5. Built-in module: `time`**

You used:

Python

```python
import time

print("Current time (seconds since 1970):", time.time())
```

- `time.time()`
    - Returns the current time as **seconds since Jan 1, 1970** (Unix timestamp).
    - Big decimal number (float).
    - Mainly used for timing things or logging.

**6. Built-in module: `math`**

You used:

Python

```python
import math

print(math.sqrt(16))  # 4.0
print(math.sqrt(25))  # 5.0
```

- `math.sqrt(x)`
    - Returns the **square root** of `x` as a float.

Other very common ones:

Python

```python
math.pi          # 3.14159...
math.floor(x)    # round down
math.ceil(x)     # round up
math.pow(a, b)   # a ** b
```

**7. Built-in module: `datetime`**

You used:

Python

```python
import datetime

print(datetime.datetime.now())
```

- `datetime.datetime.now()`
    - Returns the **current date and time**.
    - Example output: `2026-06-30 14:23:45.123456`

Conceptually:

- `datetime` (module)
- `datetime.datetime` (class/type inside module)
- `.now()` (method that gives “right now”)

For this lesson, the key thing was:

- You can import `datetime` and grab the **current timestamp** with dot notation.

**8. General pattern you learned**

The reusable mental model:

1. **Decide which toolbox you need**
    
    - Random numbers? → `random`
    - Time? → `time`
    - Math? → `math`
    - Dates/times? → `datetime`
2. **Import it once at the top**
    

Python

```python
import random
```

3. **Use dot notation to access tools**

Python

```python
random.randint(1, 6)
time.time()
math.sqrt(16)
datetime.datetime.now()
```
