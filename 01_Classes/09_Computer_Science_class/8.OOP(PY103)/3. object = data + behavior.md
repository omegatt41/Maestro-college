**Core idea: What is an object?**

> An **object** is one thing that bundles **data** (state) and **actions** (behavior) together.

**State (data / attributes)**

> **State** = information _about_ the object that can change over time.

Examples:

- Dog 🐶: name, age, color, hunger level
- Phone 📱: brand, model, battery level, storage left
- Coffee machine ☕: brand, power on/off, water level, temperature
- Traffic light 🚦: current color (red/yellow/green)

**Behavior (actions / what it does)**

> **Behavior** = what the object can _do_.

Examples:

- Dog: bark, eat, sleep, walk
- Phone: make calls, open TikTok, play games
- Coffee machine: brew coffee, heat water
- Traffic light: change to next color, start blinking

**How to tell them apart fast**

- If it sounds like a **verb/action** → usually **behavior**
- If it sounds like a **description/value** → usually **state**

# Recap:
- An **object** = **data (state)** + **actions (behavior)** together
- **State** = information about the object that can change (name, battery level, color, etc.)
- **Behavior** = what the object does (bark, brew, call, change color, etc.)
