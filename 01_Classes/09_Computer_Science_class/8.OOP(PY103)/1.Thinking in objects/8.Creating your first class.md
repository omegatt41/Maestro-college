Python

```python
class Pet:          # class definition (blueprint)
    pass

rex = Pet()        # instantiation (create object)

rex.name = "Rex"   # add attributes after creation
rex.age = 5
```

Short reminders:

- **Class** = blueprint: `class Pet:`
- **Instantiation** = create object: `rex = Pet()`
- **Attributes** = data on the object: `rex.name = "Rex"`
