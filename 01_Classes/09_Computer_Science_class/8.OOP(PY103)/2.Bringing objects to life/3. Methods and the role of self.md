## 🏗️ Class Blueprint / Definition
Class = Blueprint  for objects
method = function inside of a class that can describe what it can do
	- in a method the first parameter must be self'.
		- self = this object
		- self is only when writing the method not calling e.g 
```python
class Pet:
	def type(self):
		pass
pet1 = Pet("no self here")
```
python  auto sends pet1 as self
Pet() makes a new instance of Pet
pet.type()
```python
class Pet:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def speak(self):
        return f"My name is {self.name}"
pet1 = Pet("rex", 10)

```
